# PRD：商机管理系统

**版本**：v1.0

## 1. 背景与目标

### 背景
公司需要统一的商机管理系统，用于线索到商机、合同、开票与回款的全过程管理，并支持按区域公司分级权限。审批流程需要可配置，并通过钉钉做待办通知。

### 目标
- 建立统一的商机全流程管理能力：线索、商机、客户/联系人、合同、开票、回款、活动与任务。
- 具备区域组织树与基于角色的数据范围权限。
- 支持报价、合同、开票审批流程的可视化配置与执行。
- 支持钉钉 SSO 与组织同步，审批待办通知可触达审批人。

## 2. 范围与边界

### 范围内
- 线索、商机、客户/联系人、合同、开票、回款、活动、任务管理。
- 审批流程引擎与流程配置界面。
- 基础报表与看板（漏斗、阶段分布、合同/回款汇总）。
- 区域权限与组织管理。
- 钉钉登录与组织同步、通知。

### 范围外
- 外部 BI/数仓、复杂预测算法。
- 电子签章与合同法务系统对接。
- 钉钉内直接审批动作（仅通知）。

## 3. 角色与权限

### 角色
- 销售代表
- 销售主管
- 区域总监
- 总部管理员

### 数据范围规则
- 默认：自己 + 下级区域
- 可配置：仅本人 / 本区域 / 本区域+下级 / 全局

## 4. 业务模块

- 线索管理：新增、分配、转商机、跟进记录
- 商机管理：阶段推进、金额与时间、停留天数、挂起、归档
- 客户/联系人：基础信息维护、关联商机
- 合同/开票/回款：关联商机与客户，审批后生效
- 活动/任务：待办提醒、跟进记录

### 活动管理（Activity）

活动用于记录**已发生**的客户互动（电话/会议/拜访/邮件），作为跟进记录的来源。

- 新增/编辑/查看/删除活动记录
- 活动类型：电话/会议/拜访/邮件（可扩展）
- 关联对象：线索/商机/客户/联系人/合同（至少支持线索/商机/客户）
- 关键字段：主题、类型、发生时间、负责人、备注
- 支持按时间、类型、负责人、关联对象筛选
- 支持按商机聚合展示近期活动
- 权限：按区域 + 负责人范围过滤

### 任务管理（Task）

任务用于记录**待办事项**与未来行动提醒。

- 新增/编辑/查看/删除任务
- 状态：open/done/canceled
- 关联对象：线索/商机/客户/合同
- 关键字段：标题、截止时间、负责人、备注（可选优先级）
- 支持完成/取消并记录完成时间
- 任务提醒：到期提醒、逾期提示（站内，后续可扩展钉钉）
- 视图：我的任务、今日/本周到期筛选
- 权限：按区域 + 负责人范围过滤

## 4.1 细化设计（参考销售易）

### 客户管理（Account）
定位：客户主数据中心，承载商机、联系人、合同、开票、回款的聚合入口。

页面结构（销售易风格）：
- 客户列表页：顶部统计卡 + 左侧筛选 + 右侧列表（卡片/表格）
- 客户详情页：顶部摘要条 + Tabs（商机/联系人/合同/回款/跟进）
- 新建客户：分区表单（基础信息/业务信息/归属信息）

核心功能：
- 客户建档、编辑、合并、归档
- 关联商机、联系人、合同、回款的聚合视图
- 客户级别与价值评级可配置

关键字段：
- 客户名称、所属区域、负责人
- 客户级别、行业、企业性质、规模
- 统一社会信用代码、地址、电话、官网

筛选与排序：
- 客户级别、行业、负责人、区域、最近跟进时间
- 最近更新、创建时间、商机金额排序

权限点：
- 按区域+负责人数据范围
- 可配置“仅本人客户不可见他人客户”

### 联系人管理（Contact）
定位：客户关键联系人池，支撑跟进与合同沟通。

页面结构：
- 联系人列表页：筛选 + 列表（客户/职位/联系方式）
- 联系人详情页：基础信息 + 关联客户/商机 + 跟进记录
- 新建联系人：基础信息 + 归属客户

核心功能：
- 维护联系人信息与职位角色
- 关联客户与商机
- 跟进记录快速入口

关键字段：
- 姓名、职位、手机号、邮箱、所属客户
- 是否关键人、偏好、备注

筛选与排序：
- 客户名称、职位、负责人、最近跟进时间

权限点：
- 按区域+负责人
- 允许跨区域查看（可配置）

### 合同管理（Contract）
定位：商机成交后形成合同的主流程对象，支持审批与回款计划。

页面结构：
- 合同列表页：顶部统计 + 筛选 + 列表（状态/金额/回款进度）
- 合同详情页：合同信息 + 审批记录 + 回款计划 + 开票
- 新建合同：基础信息 + 关联商机/客户 + 回款计划

核心功能：
- 合同创建、编辑、提交审批、作废
- 回款计划管理（期次/金额/时间）
- 关联合同文件上传（后续扩展）

关键字段：
- 合同编号、合同名称、关联商机、客户名称
- 合同金额、最终结算金额、签约时间、合同状态
- 负责人、所属区域、审批状态

筛选与排序：
- 合同状态、审批状态、客户、负责人、签约时间

权限点：
- 按区域+负责人
- 合同审批节点可见范围

### 回款管理（Payment）
定位：合同回款的执行与追踪。

页面结构：
- 回款列表页：筛选 + 列表（合同/期次/金额/状态）
- 回款详情页：回款信息 + 关联合同/开票
- 新建回款：关联合同/期次

核心功能：
- 回款登记、状态变更
- 回款与合同金额对账提示（差额提示）
- 回款计划执行进度展示

关键字段：
- 关联合同、回款期次、应收金额、实收金额
- 回款日期、回款状态、负责人、所属区域

筛选与排序：
- 回款状态、合同、客户、回款日期、负责人

权限点：
- 按区域+负责人

### 开票申请（Invoice）
定位：合同回款前的开票申请与审批入口。

页面结构：
- 开票申请列表页：筛选 + 列表（合同/客户/金额/状态）
- 开票申请详情页：申请信息 + 审批记录
- 新建开票申请：关联合同/回款期次

核心功能：
- 新建申请、提交审批、驳回、作废
- 关联合同/客户/回款计划

关键字段：
- 申请编号、关联合同、客户名称
- 开票金额、税率、开票时间、申请状态
- 负责人、所属区域、审批状态

筛选与排序：
- 申请状态、审批状态、合同、客户、申请时间

权限点：
- 按区域+负责人
- 申请人仅可查看本人申请（可配置）

### 审批流程（Approval）
定位：报价/合同/开票的统一审批体系，区域分支审批。

流程配置（销售易风格）：
- 流程列表：按业务对象 + 区域展示
- 流程编辑：节点、审批人、条件分支（金额/区域/类型）
- 默认流程 + 区域覆盖

流程执行：
- 提交审批、审批通过、驳回、撤回
- 审批记录可追溯
- 钉钉待办提醒（仅通知）

审批节点能力：
- 条件分支（金额区间/区域/客户级别）
- 并行/会签（后续扩展）
- 指定审批人/角色/部门

权限点：
- 发起人、审批人、管理员不同可见范围
- 审批记录不可修改

## 5. 审批模块

- 流程配置：按区域配置审批节点与审批人
- 流程执行：提交、审批、驳回、记录
- 通知：钉钉待办提醒

## 6. 报表与看板

- 商机漏斗与阶段分布
- 合同金额与回款汇总
- 个人/区域业绩指标（基础视图）

## 7. 非功能需求

- 性能：支持中小规模并发与数据量
- 可用性：核心功能可用性>99.5%
- 安全：JWT+Session 双认证；敏感操作审计
- 可维护性：模块化设计，支持迭代升级

## 8. 外部集成

- 钉钉 SSO 登录
- 钉钉组织与人员同步
- 钉钉待办通知

## 9. 数据与权限原则

- 所有业务表包含 `region_id` 与 `owner_id`
- 查询统一按数据范围过滤
- 审批流程按区域匹配，默认可回退到全局流程
